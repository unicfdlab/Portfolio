Open-source simulation software

  • ReFRESCO - community based open-usage CFD code for the Maritime World.
  • OpenFOAM - open-source multipurpose and multiphysics CFD library based on finite volume method
  • NEK5000 - fast and scalable open-source spectral element solver for CFD
  • Nektar++ - open-source spectral/hp element framework for the development of high-performance scalable solvers for PDE
  • BEM++ - computational boundary element code to solve electrostatic, acoustic and electromagnetic problems

Open-source platform for software integration

  • DICEhub - framework for a simplified creation of user interfaces and utilities in the area of numerical simulations.
